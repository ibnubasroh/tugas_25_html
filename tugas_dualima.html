Mengenal Semantic Element Di HTML 5

  • Article

Ketika kita ingin membuat layout untuk menampilkan block artikel yang ada di website kita, tagartikel bisa kita gunakan dengan elemen header dan footer.
Karena tag article memiliki tipe block, artinya element ini memiliki lebar sesuai lebar broser jika kita tidak menentukan lebarnya.

  • Aside

Element ini bisa kita gunakan untuk melengkapi layout utama. Misalnya kita memiliki halaman blog maka kita bisa menambahkan aside dibagian bawah postingan untuk menjelaskan identitas penulis artikel.
Karena aside memiliki type sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
